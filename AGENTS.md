# Regras permanentes do projeto

- Utilizar Java 21.
- Executar o projeto e os testes pelo Maven Wrapper.
- Seguir a separação View, Control e Model descrita em `documentacao`.
- Nenhuma classe do Model pode importar Swing ou AWT.
- Preservar as funcionalidades originais.
- Realizar mudanças pequenas.
- Executar os testes antes de concluir cada etapa.
- Não fazer commits sem autorização.
